##  Project Overview

This project is a **C# console-based Cybersecurity Awareness Chatbot** designed to educate users about common cyber threats such as phishing, malware, password security, and safe browsing.

The chatbot interacts with users in a conversational manner and provides guidance on how to stay safe online. This project is inspired by research highlighting the increasing cyber threats in South Africa.



##  Purpose of the Project

The purpose of this chatbot is to:

* Educate users about cybersecurity risks
* Promote safe online behaviour
* Provide an interactive learning experience
* Raise awareness in South Africa



##  Features

*  Voice greeting using a WAV file
*  ASCII art displayed at startup
*  Personalised user interaction
*  Predefined chatbot responses
*  Input validation for invalid input
*  Enhanced console UI with colours and typing effect
*  Exit option (type `exit`)
*  GitHub version control with CI

1. Clone or download the repository
2. Open the project in Visual Studio
3. Ensure:

   * `ascii.txt` and `greeting.wav` are set to:

     * **Build Action:** Content
     * **Copy to Output Directory:** Copy if newer
4. Run the project

##  How It Works

* The program displays ASCII art at startup
* A voice greeting is played
* A text greeting is shown
* The user enters their name
* The chatbot responds to cybersecurity-related questions
* The user can exit the program by typing "exit"
